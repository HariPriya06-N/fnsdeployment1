Case Study: Personal Portfolio Website

Project Overview:

I developed a responsive personal portfolio website to showcase my projects, skills, and contact information. It was designed to be clean, simple, and professional, with easy navigation and smooth user experience.

Problem Statement:

I needed an online presence to share my projects, resume, and skills with recruiters or anyone interested in my work.

Technologies Used:

  • HTML5
  • CSS3 (Flexbox, Media Queries)
  • Font Awesome
  • GitHub for version control
  • Netlify for deployment

What I Did:

  • Structured the layout with HTML.
  • Styled the design using CSS for responsiveness and visual clarity.
  • Integrated social media icons using Font Awesome.
  • Hosted the website live using Netlify.

Challenges Faced:

Making the design fully responsive for different screen sizes and ensuring cross-browser compatibility was initially challenging.

Outcomes:

The final website is a clean, responsive, and functional portfolio that presents my skills and work clearly. This project improved my confidence in frontend development and deployment.
